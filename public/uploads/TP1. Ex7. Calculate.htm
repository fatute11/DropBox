TP1. Ex7. Calculate

Temps de référence 5 mn
Temps estimé 15 mn
Temps max 25 mn

Enoncé

Réaliser un programme permet d'effectuer les 5 opérations arithmétiques de bases, à savoir l'addition, la soustraction, la multiplication, la division entière et le modulo. Votre programme devra gérer les erreurs suivantes : le manque de paramètre, la division par 0. En cas d'erreur d'oppérateur, ils renverra expression invalide.

Exemple de résulat(s) attendu(s)

Mike 2 + 2
4
Mike 1 + -1
0
Mike 3 / 0
expression invalide
Mike 3 [espace] 0
Un des paramètres ait erroné
Mike 3 a 0
Un des paramètres ait erroné
Mike 3 +
Nombre de paramètres insuffisants
Mike 3 + 3 2
Nombre de paramètre excessif
