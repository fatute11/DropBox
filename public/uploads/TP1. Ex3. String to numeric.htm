TP1. Ex3. String to numeric

Temps de référence 5 mn
Temps estimé 15 mn
Temps max 20 mn

Enoncé

Si l'utilisateur entre moins de 2 valeurs, le message « Nombre de paramètre insuffisant » est renvoyé. Si l'utilisateur entre plus de 2 valeurs, le message « Nombre de paramètre excessif » est renvoyé. Lorsque le premier paramètre "Mike" n'est pas transmis dans l'input, le message « Le paramètre Mike est introuvable » est renvoyé. Si l'utilisateur a bien inséré Mike suivi de caractère numérique, alors afficher le message « Entier suivant : » suivi de l'entier(int) fournie. Pour tout autre type de donnée non numérique, revoyez «false» de type boolean depuis la fonction.
